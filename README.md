# AI Interview Preparation App 🚀

An AI-powered web application that analyzes a resume and job description to help candidates prepare for interviews.
The system extracts skills, calculates ATS score, and generates personalized technical and HR interview questions with answers.

## 🌐 Live Demo

🔗 https://ai-interview-prep-app-aomwbflmeyscpt2hj6sja3.streamlit.app

## ✨ Features

* Upload resume (PDF)
* Extract skills from resume and job description
* Calculate ATS (Applicant Tracking System) score
* Generate beginner → advanced technical interview questions
* Generate HR interview questions
* AI-generated answers with explanations and code
* Resume improvement suggestions

## 🛠 Tech Stack

* **Python**
* **Streamlit**
* **Groq LLM**
* **pypdf**
* **scikit-learn**

## ▶️ Run Locally

Clone repository

```
git clone https://github.com/PattiHarshini/AI-Interview-Prep-App.git
```

Install dependencies

```
pip install -r requirements.txt
```

Run the app

```
streamlit run frontend/ui.py
```
